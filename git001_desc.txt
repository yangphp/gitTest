Git是什么？
Git 是一个开源的分布式版本控制系统，用于敏捷高效地对任何或小或大的项目进行版本管理。

Git的作者？
Git 是 Linus Torvalds 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件。

Git的特点？
Git 与常用的版本控制工具 CVS, Subversion 等不同，它采用了分布式版本库的方式

什么是分布式？
是将一台计算机需要巨大计算能力才能解决的问题，分配给很多个计算机处理，存储。提高系统的可靠性，可用性和扩展性

Git和SVN的区别
git不仅仅是版本控制系统，还是内容管理系统CMS，工作管理系统等
1、git是分布式的，svn不是
2、git把内容按元数据方式存储，svn是按文件存储
3、git没有全局的版本号，而svn有
4、git的内容完整性要优于svn。git的内容存储使用sha-1哈希算法。能确保代码的完整性
